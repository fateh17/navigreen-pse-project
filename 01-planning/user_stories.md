# User story 1: As a NaviGreen user, I want to see whether I arrive at my destination most efficiently, so that the carbon footprint can be reduced.
  Acceptance criteria: Given the NaviGreen start menu, when I click the "foot-shaped" icon in the top-left corner, then the amount of carbon saved is displayed.

# User story 2: As a NaviGreen user, I want to pick the most environmentally friendly transport option, so that unnecessary emissions are not emitted.
  Acceptance criteria: Given the NaviGreen application interface, when I switch to the transportation page, then the transport options along with estimated carbon emissions are shown. 

# User story 3: As a NaviGreen user, I want to set a goal to save carbon footprints as much as possible, so that the savings plan is actually effective.
  Acceptance criteria: Given the NaviGreen application interface, when I switch to the weekly or monthly goal page, then the personalised plan that can be customised is shown.

# User story 4, As a NaviGreen user, I want to stay locked in during the whole saving plan period, so that I don't get influenced by others. 
  Acceptance criteria: Given the NaviGreen personalised plan page, when I finish the weekly or monthly plan, then the blind gift will be rewarded as an appreciation for dedication.

# User story 5, As a NaviGreen user, I want to check the estimated duration and calories burned of the specific travel option, so that it would be beneficial and practical for both me and the environment.
  Acceptance criteria: Given the NaviGreen transportation options, when I click the "details" button of the plan I choose, then the distance, duration and the burned-calories page are displayed.

# User story 6, As a NaviGreen user, I want to see the updated information regarding carbon footprints, so that I would be more aware of my contribution to climate change. 
  Acceptance criteria: Given the NaviGreen application interface, when I switch to the news page, then the facts, eco-tips and tricks, and every latest details about environment are shown.
