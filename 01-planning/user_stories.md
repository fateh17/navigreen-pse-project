# User story 1: As a NaviGreen user, I want to see whether I arrive at my destination most efficiently, so that the carbon footprint can be reduced.
  Acceptance criteria: Given the NaviGreen start menu, when I click the "foot-shaped" icon in the top-left corner, then the amount of carbon saved is displayed.

# User story 2: As a NaviGreen user, I want to pick the most environmentally friendly transport option, so that unnecessary emissions are not emitted.
  Acceptance criteria: Given the NaviGreen application interface, when I switch to the transportation page, then the transport options along with estimated carbon emissions are shown. 

# User story 3: As a NaviGreen user, I want to set a goal to save carbon footprints as much as possible, so that the savings plan is actually effective.
  Acceptance criteria: Given the NaviGreen application interface, when I switch to the weekly or monthly goal page, then the personalised plan that can be customised is shown.

# User story 4: As a NaviGreen user, I want the system to track my trips and emissions automatically, so that I can monitor my overall carbon savings. 
  Acceptance criteria: Given the NaviGreen dashboard, when I take a trip, then the system updates the total emissions saved and displays it in the "User’s History and Summary".

# User story 5: As a NaviGreen user, I want to see the estimated carbon emissions, distance, and travel time for each transport option, so that I can make the most efficient and eco-friendly choice.
  Given the NaviGreen transportation page, when I select a route, then the system displays the emissions, distance, and estimated travel time for all transport options.
  
# User story 6: As a NaviGreen user, I want to view my up to date carbon emissions for all of my trips, so that I can track my impact on the environment.
  Acceptance criteria: Given the NaviGreen dashboard, when I open the “My Emissions” page, then the system displays the cumulative carbon footprint, recent trips, and emissions for each transport option.
